# SpaceRun

Simple cave runner game written in Python using Pygame

![Game](media/game.png?raw=true)


## Controls

| Button | Function                      |
|--------|-------------------------------|
| Space  | Apply thrust, Start game      |
| Escape | Pause/unpause, Return to menu |
| Up     | Volume up                     |
| Down   | Volume down                   |
| Right  | Next song                     |
| Left   | Previous song                 |
