# Cave Run

Simple cave runner game written in Python using Pygame


## Controls

| Button | Function      |
|--------|---------------|
| Space  | Apply thrust  |
| Escape | Pause/unpause |
| Up     | Volume up     |
| Down   | Volume down   |
| Right  | Next song     |
| Left   | Previous song |
